# 📊 Portfolio Dashboard

## Быстрый старт

### 1. Установите зависимости
pip install -r requirements.txt

### 2. Получите бесплатный DeBank API ключ
Зайдите на cloud.debank.com → Sign Up → Create API Key (бесплатный план: 2000 запросов/сутки)

### 3. Вставьте ключ в dashboard.py
DEBANK_KEY = "ваш_ключ"

### 4. Запустите
streamlit run dashboard.py

Откроется браузер на http://localhost:8501

## Что показывает дашборд
- 💼 Общий баланс в $ и ₽ (реальное время)
- 🟢 Мои активы без заёмных
- 📈 P&L в долларах и рублях с %
- 🏦 Займы: сумма, стоимость/год, LTV
- 📊 Доход на заёмные деньги
- 🪙 Пирог распределения токенов
- 🔄 Авто-обновление каждые 60 секунд
